# 💰 AI Financial Dashboard

This is my full-stack project where I built a financial dashboard to analyze data and generate insights using charts and an AI assistant.

The goal of this project was to combine data analysis with web development and create something that feels like a real-world product.

---

## 🚀 What this project does

* Shows overall financial data like income, expenses, and balance
* Displays trends using charts (monthly, category-wise, payment methods)
* Allows users to ask questions about data using an AI assistant
* Supports voice input for interacting with AI
* Tracks recent transactions in a clean UI

---

## 🛠 Tech Stack

* **Frontend:** React, Recharts
* **Backend:** Node.js, Express
* **Database:** MySQL
* **AI Integration:** OpenAI API

---

## 💡 Why I built this

I wanted to practice:

* working with real datasets

This project helped me understand how data can be visualized and how users can interact with it in a smarter way.

## ⚙️ How to run this project

### 1. Clone the repo

```bash
git clone https://github.com/your-username/Financial-Dashboard.git
```

### 2. Run frontend

```bash
cd client
npm install
npm start
```

### 3. Run backend

```bash
cd server
npm install
node index.js
```

---

## 🔐 Environment setup

Create a `.env` file in backend folder:

```env
OPENAI_API_KEY=your_api_key
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=your_database
```
